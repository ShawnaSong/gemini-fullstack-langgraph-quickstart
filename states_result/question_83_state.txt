5. **Migration from requirements.txt:**
* To migrate from `requirements.txt` to `pyproject.toml` with uv:
    * Create a `pyproject.toml` file using `uv init --bare`. [pydevtools](https://vertexaisearch.cloud.google.com/id/0-9)
    * Import existing requirements using `uv add -r requirements.txt`. [pydevtools](https://vertexaisearch.cloud.google.com/id/0-9)
    * If you have development requirements, import them with `uv add --dev -r requirements-dev.txt`. [pydevtools](https://vertexaisearch.cloud.google.com/id/0-9)
    * Remove the old `requirements.txt` files. [pydevtools](https://vertexaisearch.cloud.google.com/id/0-9)
